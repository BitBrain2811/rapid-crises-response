# Crisis Response System

A web application to report and track crisis alerts.

## Project Structure
- `backend/` - Node.js/Express API connected to MongoDB.
- `frontend/` - HTML/CSS/JS frontend communicating with the API.
- `database/` - MongoDB (runs separately).

## How to run
1. Ensure MongoDB is running on `localhost:27017`.
2. Navigate to `backend` and run `npm install`.
3. Start the backend server with `npm start`.
4. Open `frontend/index.html` in your browser.
